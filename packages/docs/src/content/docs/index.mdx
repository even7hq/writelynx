WriteLynx é um motor **dual**: **valida** documentos (CPF, CNPJ, cartão com checksum) e **redige** logs e payloads (campos e padrões em texto).

## Escolha o runtime

- [JavaScript / Node](/usage/javascript/) - pacote npm `writelynx`
- [Lua / OpenResty](/usage/lua/) - módulo em `packages/lua`
- [CLI](/usage/cli/) - binário `writelynx` para pipes e arquivos
